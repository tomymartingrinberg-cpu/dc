## Instalación Rápida

Si tienes `make` y `ncurses` instalados, puedes tenerlo listo en segundos:

```bash
git clone [https://github.com/tomasmartingrinberg-cpu/dc.git](https://github.com/tomasmartingrinberg-cpu/dc.git)
cd dc
make
sudo make install
